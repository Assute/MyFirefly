做淘宝的卖家应该都有这个痛点，想快速看看哪些订单有买家的**真实评价**，而不是系统自动给的默认好评。

手动一个个点开订单去看？订单少还好，订单一多根本看不过来。

所以我写了一个小工具，帮你**批量抓取订单、自动检查评价**，把买家主动写的评价筛出来，默认好评直接过滤掉。

---

## 这个工具能做什么？

简单来说就是三步：

1. **输入商品名称和时间范围**，工具自动抓取这段时间内所有交易成功的订单
2. **逐个检查每个订单的评价**，自动判断是买家主动评价还是系统默认好评
3. **汇总结果**，告诉你哪些订单有真实评价、评价内容是什么、是好评还是中差评

### 核心功能

- **智能识别**：自动区分“买家主动评价”和“系统默认好评”，不用再一个个人工去翻
- **评价分类**：好评、中评、差评一目了然
- **可点击链接**：日志里直接生成评价详情链接，点一下就能跳转查看原始评价页面
- **日历选择**：内置日历弹窗选日期，不用手动输入
- **结果导出**：支持把结果导出为 JSON 文件，方便后续处理
- **随时停止**：运行过程中可以随时停止，不用等跑完

## 使用方法

### 方式一：下载 EXE 直接运行

推荐直接使用，不需要安装 Python，也不需要任何编程知识。

从 GitHub 的 Releases 页面下载 `淘宝评价检查工具.exe`，双击打开就能用。

### 方式二：从源码运行

如果你有 Python 环境，也可以直接跑源码：

```bash
git clone https://github.com/Assute/taobao-review-checker.git
cd taobao-review-checker
pip install requests
python 淘宝评价检查工具.py
```

### 操作步骤

1. 打开工具，输入你要查询的**商品名称**
2. 选择**起止日期**，默认是三个月前
3. 打开浏览器登录淘宝卖家后台，按 `F12` 获取 **Cookie**，也可以使用 [Cookie Viewer & Exporter 插件](/posts/cookie-viewer-exporter/) 一键获取，粘贴到工具里
4. 点击“开始运行”
5. 等待运行完成，查看结果

工具会先分页抓取所有交易成功的订单，然后逐个去检查评价状态。过程中会实时显示进度和日志，跑完后给出汇总报告。

## 运行效果

工具运行完成后，会输出类似这样的汇总：

```text
============================================================
汇总结果
============================================================
总订单数: 86
买家主动评价: 12
系统默认好评: 68
无评价: 4
请求失败: 2
```

对于有买家主动评价的订单，还会显示评价内容、评价类型（好评 / 中评 / 差评）和评价时间，并附上可直接点击的详情链接。

## 技术实现

工具基于 Python 开发，使用 `tkinter` 做图形界面，`requests` 发起 HTTP 请求。

主要流程：

1. 调用淘宝已卖出商品接口，按商品名称和时间范围筛选，分页获取所有交易成功的订单号
2. 对每个订单号，请求淘宝评价详情页面，解析 HTML 提取评价内容
3. 通过关键词匹配，例如“系统默认好评”“此用户没有填写评价”等，判断是否为真实评价
4. 汇总统计并展示结果
