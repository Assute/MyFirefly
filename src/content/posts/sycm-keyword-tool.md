做电商运营的朋友都知道，生意参谋是淘宝卖家最重要的数据分析工具之一。而其中的搜索词分析功能，更是选品、优化标题、直通车投放的核心数据来源。

但问题来了，手动一个个类目去查关键词，效率实在太低了。

淘宝有几十个一级类目，每个下面还有二级、三级类目，全部手动采集一遍，少说也要好几天。更别说你还需要定期更新数据了。

**所以我开发了这款工具，帮你自动完成这件事。**

---

## 一、这个工具能做什么？

简单来说，它可以自动登录生意参谋，按类目层级批量采集搜索关键词数据，最后导出为 Excel 文件。

具体功能包括：

1. **自动化采集**：通过 Selenium 自动操控 Chrome 浏览器，模拟人工操作，从生意参谋页面批量提取搜索关键词数据。
2. **多级类目遍历**：支持一级、二级、三级类目的自动遍历，不用手动一个个点。
3. **智能筛选**：可以设置搜索人气阈值，默认 `150`，自动过滤掉没有价值的长尾词，只保留有搜索量的关键词。
4. **自动翻页**：每个类目最多采集 6 页数据，自动翻页，不需要人工干预。
5. **Excel 导出**：采集到的数据自动保存为 Excel 文件，按类目分类整理，拿来就能用。
6. **暂停 / 继续**：采集过程中可以随时暂停和继续，不怕中途有事要离开。
7. **多浏览器并行**：支持多个浏览器实例同时采集，大幅提升效率。
8. **图形界面**：基于 tkinter 开发了可视化操作界面，不需要懂代码也能用。

## 二、怎么使用？

### 方式一：直接下载 EXE 运行

推荐小白用户使用。我已经把程序打包成了 exe 文件，不需要安装 Python 环境，双击就能运行。

**下载地址：**

- GitHub: <https://github.com/Assute/sycm-keyword-tool/releases>
- Gitee: <https://gitee.com/Assute/sycm-keyword-tool/releases>

### 方式二：从源码运行

适合有 Python 基础的用户。

1. 安装 Python 3.10 以上版本
2. 安装依赖：

```bash
pip install selenium openpyxl pywin32 webdriver-manager
```

3. 运行：

```bash
python 生意参谋关键词获取工具.py
```

## 三、使用步骤

| 步骤 | 操作说明 |
| --- | --- |
| 第一步 | 启动程序，打开工具界面 |
| 第二步 | 在程序打开的 Chrome 浏览器中登录生意参谋账号 |
| 第三步 | 选择你想要采集的目标类目层级 |
| 第四步 | 点击开始采集，程序自动遍历所有类目，逐页提取关键词数据 |
| 第五步 | 采集完成后，数据自动导出为 Excel 文件，打开即可使用 |

整个过程中你可以随时暂停或停止，非常灵活。

## 四、可自定义配置

如果你有一定的编程基础，还可以在源码中调整以下参数：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `min_popularity_threshold` | `150` | 搜索人气筛选阈值，低于此值的关键词会被过滤 |
| `max_pages` | `6` | 每个类目最大采集页数 |
| `exclude_level1_serials` | `[4,34,52,53,54,58,59,60]` | 排除不需要采集的一级类目序号 |
