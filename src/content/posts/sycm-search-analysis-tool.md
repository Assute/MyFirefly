做电商选词和测词时，很多人都会用到生意参谋里的搜索分析功能。

问题在于：**单个词手动查不难，词一多就很累。**

尤其是当你已经整理好一批种子词，想逐个查看相关搜索词、搜索人气、支付买家数、需求供给比这些指标时，手动复制、翻页、导出，效率会非常低。

所以我把这款工具也单独整理成一篇文章。它更适合的场景不是“按类目漫游采集”，而是：**你已经有一批关键词，想让软件按 Excel 批量跑完整个搜索分析流程。**

## 软件是做什么的？

这款工具本质上是一套 **生意参谋搜索词深度分析自动化助手**。

从代码逻辑来看，它主要完成下面几件事：

1. 连接本地 `Chrome` 调试端口
2. 打开并读取 `Excel` 关键词表
3. 逐行取出待分析的搜索词
4. 自动进入生意参谋搜索分析页面
5. 抓取相关搜索词及核心指标
6. 把结果持续写回 `Excel`
7. 任务完成或异常时发送企业微信通知

也就是说，它更像一个“**关键词批处理执行器**”，而不是一个简单的单次查询工具。

## 适合什么人用？

如果你平时的工作流程是这样的：

- 先在表格里整理一批关键词
- 再逐个到生意参谋里查词
- 把相关搜索词和指标抄回表格

那这款工具就很适合你。

它尤其适合：

- 需要批量测词的淘宝 / 天猫运营
- 做选品前关键词筛查的人
- 需要沉淀关键词库的人
- 想把“手工查词”改成“半自动批处理”的用户

## 这款工具有哪些功能？

结合你给出的源码，我整理出它的核心能力大概有这些：

### 1）批量读取 Excel 关键词

程序会从 `Excel` 里按行读取关键词，并逐行执行搜索分析。

从代码逻辑看：

- 第 `1` 个工作表用来放待处理关键词
- `A` 列是搜索词
- `F` 列作为“是否已搜索”的完成标记

也就是说，只要你把关键词先整理到表里，它就能按行自动跑。

### 2）自动进入生意参谋搜索分析页

工具会自动拼接搜索分析页面地址，并带上日期参数，直接打开对应关键词的分析页面。

代码里默认把查询日期设置成“昨天”，这一点很适合日常运营分析。打开软件后，日期会自动填成前一天，不用每次手改。

### 3）自动抓取核心指标

从抓取逻辑来看，它会提取并保存以下几类数据：

- 相关搜索词
- 搜索人气
- 支付转化率
- 支付买家数
- 需求供给比
- 天猫商品点击占比

这些都是做搜索词判断时非常常用的指标。

### 4）结果自动写回 Excel

抓到数据以后，程序会把结果写入第 `2` 个工作表。

从代码里可以看出，第二个工作表大致会按下面结构输出：

- `A` 列：相关搜索词
- `B` 列：搜索人气
- `C` 列：支付转化率
- `D` 列：支付买家数
- `E` 列：需求供给比
- `F` 列：天猫商品点击占比

这样做的好处是：

- 原始关键词和结果分开
- 后续筛选、排序、透视更方便
- 跑完就能直接继续做分析

### 5）支持翻页采集

很多关键词的相关搜索词不只一页。

这套工具会在页面里继续点“下一页”，把后续结果也抓下来，而不是只拿首页数据。对于想做更完整关键词池的人来说，这点很实用。

### 6）支持风险控制后的延迟重试

源码里专门写了滑块检测逻辑。

也就是说，如果页面出现风控验证，程序不会直接崩掉，而是会：

- 记录日志
- 暂停当前关键词处理
- 延时后再重试当前词

这个设计的意义很大，因为批量任务最怕的就是跑到一半中断。

### 7）支持 Cookie 保存与恢复登录

工具里还有两个很实用的小功能：

- 保存当前登录 Cookie
- 读取本地 Cookie 重新登录

对于经常跑批量任务的人来说，这能省掉一部分重复登录操作。

### 8）支持企业微信通知

从源码里还可以看到，它支持企业微信消息推送。

这意味着：

- 任务跑完可以通知你
- 出现异常时也可以提醒你

如果你经常把任务挂着跑，这个功能就很实用，不需要一直盯着电脑看。

## 这款工具怎么使用？

我按它现在的设计思路，整理出一套比较顺手的使用流程。

### 第一步：准备 Excel 关键词表

先准备一个 Excel 文件。

建议按下面思路整理：

- 第 `1` 个工作表放待分析关键词
- `A` 列填写搜索词
- `F` 列留空，作为任务完成标记
- 第 `2` 个工作表用来接收输出结果

这样程序在执行时会比较顺。

### 第二步：启动浏览器并登录生意参谋

软件支持通过指定端口启动 `Chrome`，然后连接浏览器。

正常使用时，你可以：

1. 填好浏览器路径
2. 填好调试端口
3. 点击“启动浏览器”
4. 在打开的浏览器里手动登录生意参谋

如果你已经登录过，也可以根据自己的习惯使用它的 `Token 保存 / Token 登录` 功能。

### 第三步：打开 Excel

在软件里选择 Excel 文件路径，然后点击“启动 Excel”。

程序会读取表格内容，并根据设置决定 Excel 是否可见。

如果你想一边看表格一边跑任务，就把 Excel 可视打开；如果想后台运行，就可以关闭可视显示。

### 第四步：设置日期和间隔

软件启动后默认会把日期设成昨天。

你也可以手动修改成自己想查的日期。

另外它还支持设置：

- 搜索间隔时间
- 翻页间隔时间

这两个参数主要影响整体执行节奏。

### 第五步：点击开始执行

点击“开始”后，程序会按行读取关键词，并开始自动抓取。

抓取过程中会把完成状态写回原表，同时把结果写到第二个工作表。

如果中途需要停下来，也可以点击停止。程序会先保存当前 Excel，再等待当前任务收尾。

## 总结

如果说前一类工具更偏向“批量采集类目关键词”，那这款软件更偏向“**批量跑搜索词深度分析**”。

它的核心优势在于：

- 直接读取现成关键词表
- 自动进入分析页
- 自动翻页抓取指标
- 自动写回 Excel
- 支持登录状态复用和消息提醒

对于经常做关键词分析、竞品词筛选、相关词池沉淀的人来说，这类工具确实能省掉很多重复操作。

如果后面你还想继续把这个工具完善下去，我觉得最值得继续打磨的方向有三个：

1. 增加更多指标列的自定义开关
2. 增加任务失败重跑记录
3. 增加结果去重和筛选导出能力

这样它会更像一个完整的运营分析工作台，而不只是一个批量执行小工具。
